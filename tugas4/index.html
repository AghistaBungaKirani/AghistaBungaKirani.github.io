  • Arti: "Banyak bentuk" (poly = banyak, morph = bentuk).
  • Konsep: Objek dari subclass dapat diperlakukan sebagai objek dari superclass.

🎀 Manfaat Polimorfisme

  • Fleksibilitas: Kode dapat bekerja dengan berbagai tipe.
  • Extensibility: Mudah menambah tipe baru.
  • Code Reusability: Method umum di superclass.
  • Maintainability: Perubahan mudah dikelola.

🎀 is dan as Operator

  • • is: Mengecek tipe objek.
  • • as: Casting tipe objek.

void main() {
  Kendaraan k = Mobil();

  // Menggunakan is (cek tipe)
  if (k is Mobil) {

  }

  // Safe casting dengan as?
  (k as Mobil?)?.klakson();
}

4.Tablet – Multiple Inheritance dengan implements/mixins(Bonus 2)

Class Tablet mewarisi kemampuan Komputer, Kamera, Telepon (multiple inheritance disimulasikan dengan implements).

DART SOURCE CODE
//Nama: Aghista Bunga Kirani
//Nim: 251410027
//Kelas: SI2A


// Superclass
class Komputer {
  void prosesData() {
    print('Komputer: Memproses data 💻');
  }
}

// "Interface"
class Kamera {
  void ambilFoto() {}
}

class Telepon {
  void telepon() {}
}

// Subclass Tablet
class Tablet extends Komputer implements Kamera, Telepon {
  @override
  void ambilFoto() {
    print('Tablet: Mengambil foto 📸');
  }

  @override
  void telepon() {
    print('Tablet: Menelepon 📞');
  }
}

void main() {
  Tablet tablet = Tablet();

  tablet.prosesData(); // dari Komputer
  tablet.ambilFoto();  // dari Kamera
  tablet.telepon();    // dari Telepon
}

5.MathUtils – Static Methods (Faktorial, Prima, Random, Rata2, Median)(Bonus 3)

DART SOURCE CODE
//Nama: Aghista Bunga Kirani
//Nim: 251410027
//Kelas: SI2A


import 'dart:math';

class MathUtils {
  // 1. Faktorial
  static int factorial(int n) {
    if (n < 0) throw Exception('Tidak boleh negatif');
    int hasil = 1;
    for (int i = 1; i <= n; i++) {
      hasil *= i;
    }
    return hasil;
  }

  // 2. Cek bilangan prima
  static bool isPrime(int n) {
    if (n <= 1) return false;
    for (int i = 2; i <= sqrt(n); i++) {
      if (n % i == 0) return false;
    }
    return true;
  }

  // 3. Membulatkan double ke desimal tertentu
  static double roundDouble(double value, int places) {
    double mod = pow(10.0, places).toDouble();
    return (value * mod).round() / mod;
  }

  // 4. Random integer dalam range [min, max]
  static int randomInt(int min, int max) {
    return min + Random().nextInt(max - min + 1);
  }

  // 5. Random double dalam range [min, max]
  static double randomDouble(double min, double max) {
    return min + Random().nextDouble() * (max - min);
  }

  // 6. Rata-rata (mean)
  static double mean(List data) {
    if (data.isEmpty) throw Exception('List kosong');
    return data.reduce((a, b) => a + b) / data.length;
  }

  // 7. Median
  static double median(List data) {
    if (data.isEmpty) throw Exception('List kosong');

    List sorted = List.from(data)..sort();
    int n = sorted.length;

    if (n % 2 == 1) {
      return sorted[n ~/ 2].toDouble();
    } else {
      return (sorted[n ~/ 2 - 1] + sorted[n ~/ 2]) / 2;
    }
  }
}

void main() {
  print('Faktorial 5: ${MathUtils.factorial(5)}');
  print('7 prima? ${MathUtils.isPrime(7)}');
  print('Pembulatan: ${MathUtils.roundDouble(3.14159, 2)}');
  print('Random int: ${MathUtils.randomInt(1, 10)}');
  print('Random double: ${MathUtils.randomDouble(1, 5)}');
  
  List angka = [1, 2, 3, 4, 5];
  print('Mean: ${MathUtils.mean(angka)}');
  print('Median: ${MathUtils.median(angka)}');
}

🎀 Abstract Class

Abstract class adalah class yang: ❌ Tidak bisa dibuat objek langsung ✅ Digunakan sebagai kerangka dasar ✅ Bisa punya: • Method abstract → tanpa isi (harus dioverride) • Method concrete → ada isi (bisa langsung dipakai).

DART SOURCE CODE

//Nim: 251410027

abstract class Kendaraan {
  String merk;

  Kendaraan(this.merk);

  // Method abstract
  void start();

  // Method concrete
  void stop() {
    print('Kendaraan berhenti');
  }
}

class Mobil extends Kendaraan {
  Mobil(String merk) : super(merk);

  @override
  void start() {
    print('Mobil $merk menyala 🚗');
  }
}

void main() {
  // Kendaraan k = Kendaraan(); ❌ ERROR (tidak bisa diinstansiasi)

  Mobil mobil = Mobil('Toyota');

  mobil.start(); // method dari override
  mobil.stop();  // method dari abstract class
}

🎀 Abstract Method

Abstract method adalah method yang hanya dideklarasikan (tanpa badan/implementasi) di dalam abstract class.

DART SOURCE CODE
//Nama: Aghista Bunga Kirani
//Nim: 251410027
//Kelas: SI2A


// Abstract class
abstract class AlatMusik {
  // Abstract method (tanpa body)
  void mainkan();

  // Concrete method (sudah ada implementasi)
  void info() {
    print('Ini adalah alat musik');
  }
}

// Subclass yang mengimplementasikan abstract method
class Gitar extends AlatMusik {
  @override
  void mainkan() {
    print('Jreng jreng...');
  }
}

class Drum extends AlatMusik {
  @override
  void mainkan() {
    print('Dug dug dug...');
  }
}

void main() {
  // AlatMusik alat = AlatMusik(); // ERROR: abstract class tidak bisa diinstansiasi
  
  Gitar gitar = Gitar();
  gitar.mainkan(); // Output: Jreng jreng...
  gitar.info();    // Output: Ini adalah alat musik
  
  print('---');
  
  Drum drum = Drum();
  drum.mainkan(); // Output: Dug dug dug...
  drum.info();    // Output: Ini adalah alat musik
}

🎀 Interface di Dart

  • :• Sejak Dart 3.0, Dart memiliki keyword interface class untuk mendefinisikan interface secara eksplisit.
  • :• Semua class sebenarnya sudah bisa menjadi implicit interface (tanpa keyword interface).
  • :• Ketika menggunakan implements, method dari interface class tidak diwarisi (harus diimplementasi ulang).
    • DART SOURCE CODE
    //Nama: Aghista Bunga Kirani
//Nim: 251410027
//Kelas: SI2A


// Interface class (Dart 3.0 ke atas)
interface class Flyable {
  void terbang() {
    print('Terbang...');
  }
  
  void landing() {
    print('Mendarat...');
  }
}

// Class mengimplementasi interface
class Bird implements Flyable {
  @override
  void terbang() {
    print('Burung terbang tinggi sekali');
  }
  
  @override
  void landing() {
    print('Burung hinggap di pohon');
  }
  
  // Method tambahan milik Bird sendiri
  void bernyanyi() {
    print('Cuit cuit...');
  }
}

class Plane implements Flyable {
  @override
  void terbang() {
    print('Pesawat terbang dengan kecepatan 900 km/jam');
  }
  
  @override
  void landing() {
    print('Pesawat mendarat di bandara');
  }
}

// Contoh class yang mengimplementasi interface
class Drone implements Flyable {
  @override
  void terbang() {
    print('Drone terbang diam di udara');
  }
  
  @override
  void landing() {
    print('Drone mendarat perlahan');
  }
}

void main() {
  print('=== INTERFACE DI DART ===\n');
  
  // Membuat objek dari masing-masing class
  Bird burung = Bird();
  Plane pesawat = Plane();
  Drone drone = Drone();
  
  print('-- Burung --');
  burung.terbang();
  burung.landing();
  burung.bernyanyi();
  
  print('\n-- Pesawat --');
  pesawat.terbang();
  pesawat.landing();
  
  print('\n-- Drone --');
  drone.terbang();
  drone.landing();
  
  print('\n--- POLIMORFISME DENGAN INTERFACE ---');
  
  // Polymorphism: semua bisa diperlakukan sebagai Flyable
  List daftarTerbang = [burung, pesawat, drone];
  
  for (var obj in daftarTerbang) {
    obj.terbang();
  }
}

    🎀 Static Members dan Method

  • :• Static members adalah properti atau method yang dimiliki oleh class itu sendiri, bukan oleh objek class. Static member diakses langsung melalui class name.
  • :• Properti dan method static sangat berguna untuk membuat utility class atau menyimpan konstanta yang terkait dengan class, sehingga dapat diakses langsung tanpa perlu membuat objek.
    • DART SOURCE CODE
    //Nama: Aghista Bunga Kirani
//Nim: 251410027
//Kelas: SI2A


// ========== STATIC MEMBERS - CONTOH SEDERHANA ==========

class MathUtils {
  // Static property (milik class)
  static double pi = 3.14;
  
  // Static method (milik class)
  static int tambah(int a, int b) {
    return a + b;
  }
}

void main() {
  // ✅ Akses langsung lewat NAMA CLASS (tanpa membuat objek)
  print('Nilai PI: ${MathUtils.pi}');
  print('10 + 20 = ${MathUtils.tambah(10, 20)}');
  
  // ❌ Ini ERROR! Tidak bisa akses static lewat objek
  // var math = MathUtils(); // Tidak perlu!
  // print(math.pi); // ERROR!
}
