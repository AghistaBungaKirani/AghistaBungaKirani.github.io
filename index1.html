Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berfokus pada objek sebagai komponen utama dalam membangun sebuah program. Dalam OOP, program dibangun dari objek-objek yang memiliki data (state) dan perilaku (behavior). Tujuannya agar kode lebih terstruktur, mudah dipahami, mudah dikembangkan, serta dapat digunakan kembali.

Object, Class, dan Konsep Dasar

Class adalah cetakan atau rancangan untuk membuat object. Object adalah hasil dari class yang memiliki atribut (data) dan metode (fungsi). Class menentukan bagaimana object dibuat dan bagaimana object tersebut bekerja.

Class Sederhana (Car)

Contoh class sederhana adalah class Car yang memiliki atribut seperti merk dan warna, serta metode seperti jalan() dan berhenti(). Object dari class ini dapat dibuat untuk merepresentasikan mobil yang berbeda-beda.
