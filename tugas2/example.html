🚗 slide 3

Named Constructor

import 'dart:convert'; class Mobil { String? merk; int? tahun; Mobil(this.merk, this.tahun); // parameterized Mobil.baru() { // named merk = 'Toyota'; tahun = 2023; } Mobil.dariJson(Map<String, dynamic> json) { // named + json merk = json['merk']; tahun = json['tahun']; } @override String toString() => 'Mobil{merk: $merk, tahun: $tahun}'; }

💕 panggil: Mobil.baru() atau Mobil.dariJson(data)

🧊 slide 4

Constant Constructor

immutable semua field harus final

class Warna { final int r, g, b; const Warna(this.r, this.g, this.b); @override String toString() => 'Warna(r: $r, g: $g, b: $b)'; } void main() { const merah = Warna(255, 0, 0); const merahLagi = Warna(255, 0, 0); // instance SAMA print(merah); print(merahLagi); }

🌸 hemat memori, cocok untuk warna, koordinat, dll.

🔒 slide 7

Enkapsulasi

  • menyembunyikan detail implementasi
  • tujuan: keamanan, kontrol, fleksibilitas
  • di Dart: pakai underscore _ private
💗 privat hanya dalam file yang sama
👀 slide 8

Public vs Private

class Person { String _name; // private (hanya file ini) int age; // public (default) Person(this._name, this.age); }

⭐ _name tidak bisa diakses dari file lain.

🔍 slide 9

Getter (mengakses private)

class Person { String _name; int age; Person(this._name, this.age); String get getName1 => _name; // getter style 1 String getName() => _name; // method style 2 }
✏️ slide 10

Setter (memodifikasi private)

class Person { String _name; int age; Person(this._name, this.age); set setName1(String name) => _name = name; // setter void setName(String name) => _name = name; // method versi lain }
🛍️ slide 11

Getter & Setter + Validasi (Produk)

class Produk { String _nama; double _harga; Produk(this._nama, this._harga); String get nama => _nama; double get harga => _harga; set nama(String value) { if (value.length >= 3) { _nama = value; } else { print('❌ nama minimal 3 karakter'); } } set harga(double value) { if (value > 0) { _harga = value; } else { print('❌ harga harus positif'); } } }

🧸 data aman, tidak bisa sembarang diubah.

📝 slide 12

Praktik 2 · Enkapsulasi Lengkap (User)

Buat class dengan field private: _username, _password, _email + getter & setter + validasi:

  • username minimal 5 karakter
  • password minimal 8 karakter
  • email harus mengandung '@'
🎀 coba modifikasi dengan setter — validasi aktif!
📱 slide 13

Studi Kasus · Login Sederhana

class User { String _username; String _password; User(this._username, this._password); String get username => _username; bool login(String username, String password) { return _username == username && _password == password; } void gantiPassword(String lama, String baru) { if (_password == lama) { if (baru.length >= 8) { _password = baru; print('✅ password berhasil diubah.'); } else { print('❌ password baru minimal 8 karakter.'); } } else { print('❌ password lama salah.'); } } }

✨ method gantiPassword dengan validasi aman ✨
